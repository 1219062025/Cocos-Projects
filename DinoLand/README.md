# 项目信息

cocos版本3.5.2

## 项目结构

```text
assets
│
├── resources   # 动态资源包，包括需要动态加载的动画、预制体、配置表json、纹理等等
│
├── scripts   # 脚本
│   └── Main.ts   # ***项目入口程序***
│   └── framework   # ***框架相关***
│       └── App.ts    # ***框架入口，启动框架，初始化各个框架管理层信息，监听游戏事件***
│       └── AppConfig.ts    # 项目客户端配置信息
│       └── Root.ts    # 根组件
│       └── common    # 框架通用。音频、时间、资源、日志等等
│           └── audio    # 全局音频管理器
│           └── event    # 全局事件管理器
│           └── log    # 全局日志管理
│           └── manager
│               └── ResourceManager    # 全局资源管理器
│               └── TimerManager    # 全局时间管理器
│           └── queue    # 异步队列控制器
│           └── storage    # 本地存储管理器
│           └── types    # 框架类型声明
│           └── utils    # 工具函数
│       └── game    # 框架三维游戏世界相关
│       └── gui   # 框架二维游戏世界相关
│           └── layer    # 2维UI视图管理层
│       └── Root.ts   # 项目根组件
│   └── game    # 游戏世界代码
│       └── data    # 场景内各个模块使用的数据信息
│       └── eventTask    # 场景事件，例如收银、升级展台等等场景内的事件都在这里
│       └── guide    # 游戏引导
│       └── manager    # 场景内的管理层
│       └── objs    # 场景内各个对象的控制脚本
│       └── scene    # 游戏场景控制，控制当前所在地图，生成玩家、恐龙、展台、NPC等等都在这里
│       └── table    # 场景内的各种配置表
│       └── ui    # 场景内的UI相关，各种血条，弹窗，buff，等级的UI等等都在这里
│   └── uniSdk    # 微信SDK
│           
├── sound   # 声音相关
│
└── ui   # UI相关的图片纹理、预制体、字体、图集等
